# ftp-termux

## INSTALACION

Inicia instalando python3 
```sh
pkg install python
```
FTP por python requiere un modulo pip llamado pyftpdlib 
```sh
python -m pip install pyftpdlib
```
```sh
pkg install nano
```
```sh
termux-setup-storage
```
Inicie ftpserver.py
```sh 
python3 ftpserver.py
```
Para poner en segundo plano 
```sh
python3 ftpserver.py &
```
